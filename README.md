

**1. users (Пользователи)**
```
+----------------+------------------+---------------------------------------------------+
| Поле           | Тип              | Описание                                          |
+----------------+------------------+---------------------------------------------------+
| id             | SERIAL PRIMARY   | Уникальный идентификатор                          |
| name           | VARCHAR(100)     | Полное имя пользователя                           |
| email          | VARCHAR(100)     | Уникальный email (UNIQUE)                         |
| password_hash  | VARCHAR(200)     | Хеш пароля                                        |
| role           | VARCHAR(20)      | Роль (admin/user)                                 |
| created_at     | TIMESTAMP        | Дата регистрации (DEFAULT CURRENT_TIMESTAMP)      |
| last_login     | TIMESTAMP        | Дата последнего входа                             |
+----------------+------------------+---------------------------------------------------+
```

**2. projects (Проекты)**
```
+----------------+------------------+---------------------------------------------------+
| Поле           | Тип              | Описание                                          |
+----------------+------------------+---------------------------------------------------+
| id             | SERIAL PRIMARY   | Уникальный идентификатор                          |
| name           | VARCHAR(255)     | Название проекта                                  |
| description    | TEXT             | Описание проекта                                  |
| owner_id       | INT REFERENCES   | Владелец проекта (users.id)                       |
| created_at     | TIMESTAMP        | Дата создания (DEFAULT CURRENT_TIMESTAMP)         |
| deadline       | DATE             | Крайний срок проекта                              |
+----------------+------------------+---------------------------------------------------+
```

**3. tasks (Задачи)**
```
+----------------+------------------+---------------------------------------------------+
| Поле           | Тип              | Описание                                          |
+----------------+------------------+---------------------------------------------------+
| id             | SERIAL PRIMARY   | Уникальный идентификатор                          |
| title          | VARCHAR(255)     | Заголовок задачи                                  |
| description    | TEXT             | Подробное описание                                |
| status         | VARCHAR(20)      | Статус (new/in_progress/completed)                |
| priority       | VARCHAR(10)      | Приоритет (high/medium/low)                       |
| due_date       | DATE             | Срок выполнения                                   |
| created_at     | TIMESTAMP        | Дата создания (DEFAULT CURRENT_TIMESTAMP)         |
| project_id     | INT REFERENCES   | Проект (projects.id ON DELETE CASCADE)            |
| assignee_id    | INT REFERENCES   | Исполнитель (users.id ON DELETE SET NULL)         |
| reporter_id    | INT REFERENCES   | Автор задачи (users.id)                           |
+----------------+------------------+---------------------------------------------------+
```

**4. task_history (История изменений)**
```
+----------------+------------------+---------------------------------------------------+
| Поле           | Тип              | Описание                                          |
+----------------+------------------+---------------------------------------------------+
| id             | SERIAL PRIMARY   | Уникальный идентификатор                          |
| task_id        | INT REFERENCES   | ID задачи (tasks.id ON DELETE CASCADE)            |
| field_name     | VARCHAR(50)      | Измененное поле                                   |
| old_value      | TEXT             | Старое значение                                   |
| new_value      | TEXT             | Новое значение                                    |
| changed_by     | INT REFERENCES   | Кто изменил (users.id)                            |
| changed_at     | TIMESTAMP        | Дата изменения (DEFAULT CURRENT_TIMESTAMP)        |
+----------------+------------------+---------------------------------------------------+
```

**5. comments (Комментарии)**
```
+----------------+------------------+---------------------------------------------------+
| Поле           | Тип              | Описание                                          |
+----------------+------------------+---------------------------------------------------+
| id             | SERIAL PRIMARY   | Уникальный идентификатор                          |
| content        | TEXT             | Текст комментария                                 |
| task_id        | INT REFERENCES   | ID задачи (tasks.id ON DELETE CASCADE)            |
| author_id      | INT REFERENCES   | Автор (users.id)                                  |
| created_at     | TIMESTAMP        | Дата создания (DEFAULT CURRENT_TIMESTAMP)         |
+----------------+------------------+---------------------------------------------------+
```

**6. tags (Теги)**
```
+----------------+------------------+---------------------------------------------------+
| Поле           | Тип              | Описание                                          |
+----------------+------------------+---------------------------------------------------+
| id             | SERIAL PRIMARY   | Уникальный идентификатор                          |
| name           | VARCHAR(50)      | Название тега (UNIQUE)                            |
| color          | VARCHAR(7)       | Цвет в HEX формате                                |
+----------------+------------------+---------------------------------------------------+
```

**7. task_tags (Связь задач и тегов)**
```
+----------------+------------------+---------------------------------------------------+
| Поле           | Тип              | Описание                                          |
+----------------+------------------+---------------------------------------------------+
| task_id        | INT REFERENCES   | ID задачи (tasks.id ON DELETE CASCADE)            |
| tag_id         | INT REFERENCES   | ID тега (tags.id ON DELETE CASCADE)               |
+----------------+------------------+---------------------------------------------------+
```
